# Asset Credits

Topogenesis uses free CC0 runtime assets from Quaternius for the Godot RPG
vertical slice.

## Included Runtime Packs

- Medieval Village MegaKit Standard by Quaternius
- Fantasy Props MegaKit Standard by Quaternius
- Stylized Nature MegaKit Standard by Quaternius
- Ultimate Animated Character Pack by Quaternius

The included pack license files declare:

```text
CC0 1.0 Universal (CC0 1.0)
Public Domain Dedication
https://creativecommons.org/publicdomain/zero/1.0/
```

Only runtime-needed formats are kept in the repository:

- `glTF`, `bin`, and local texture files for environment, prop, and nature assets
- `FBX` rigged character files for animated character variants

Downloaded zip archives and duplicate source formats are intentionally excluded
from the committed runtime set.
